# 1. 서비스 한 줄 정의

## 한 줄 요약
> **커플/부부/가족이 서로에게 칭찬 스티커를 주고, 목표를 달성하면 약속한 보상을 실행하는 관계형 리워드 앱**

## 해결하는 문제

| 문제 | 설명 |
|------|------|
| 관계 속 칭찬의 휘발성 | 말로 하는 칭찬은 기록이 안 되고 잊혀짐 |
| 동기부여 부족 | "운동해", "설거지해"만 반복 → 갈등 유발 |
| 약속 불이행 | "다음에 맛있는 거 사줄게" → 흐지부지 |
| 관계 관리 도구 부재 | 커플앱은 기념일/채팅 위주, 행동 강화 앱 없음 |

## 핵심 타겟

| 순위 | 타겟 | 비중(예상) | 시나리오 |
|------|------|-----------|---------|
| 1순위 | 20~30대 커플/부부 | 60% | 가사분담, 다이어트, 운동 등 서로 응원 |
| 2순위 | 부모-자녀 | 25% | 숙제, 심부름, 생활습관 칭찬 |
| 3순위 | 친구/룸메이트 | 15% | 공동생활 규칙, 챌린지 |

## 왜 사람들이 쓸까?

1. **게임화된 칭찬** → 스티커를 모으는 재미 (수집 본능)
2. **확실한 보상** → 목표 달성 = 약속 이행 강제
3. **패널티 옵션** → 긴장감과 재미 동시에
4. **커스텀 스티커** → 우리만의 언어/밈 공유
5. **관계 기록** → "올해 서로 칭찬 127번 했어" 같은 회고
