# 3. 기능 정의서

## MVP 필수 기능 (1차 출시)

| # | 기능 | 상세 설명 | 구현 방식 |
|---|------|---------|----------|
| F1 | 소셜 로그인 | 카카오/구글/애플 로그인. 별도 이메일 가입 없음 | Supabase Auth + OAuth Provider |
| F2 | 프로필 설정 | 닉네임, 프로필 사진 | Supabase Storage + users 테이블 |
| F3 | 관계 연동 | 6자리 초대코드 생성 → 공유 → 입력 → 수락 | 초대코드는 서버에서 랜덤 생성, 24시간 만료 |
| F4 | 스티커판 생성 | 이름, 대상, 목표 수, 기간, 보상텍스트 설정 | sticker_boards 테이블에 INSERT |
| F5 | 스티커판 수락 | 상대방이 스티커판 확인 후 수락해야 활성화 | 상태: draft → active |
| F6 | 스티커 지급 | "스티커 주기" 버튼으로 1개씩 부여, 메모 선택 | stickers 테이블에 INSERT |
| F7 | 진행률 표시 | 현재/목표 스티커 수, 퍼센트, 남은 기간 표시 | 클라이언트에서 계산 |
| F8 | 목표 달성 처리 | 스티커 수 ≥ 목표 수 → 상태 자동 변경 | 스티커 INSERT 시 트리거 또는 클라이언트 체크 |
| F9 | 기간 만료 처리 | 기간 초과 시 자동 종료 | 서버 Cron Job (매일 자정) |
| F10 | 성공 보상 표시 | 달성 시 미리 설정한 보상 텍스트 표시 | rewards 테이블 조회 |
| F11 | 보상 완료 확인 | 보상 준 사람이 "완료" 버튼 클릭 | reward status: pending → completed |
| F12 | 실패 패널티 설정 | ON/OFF 토글 + 패널티 내용 텍스트 입력 | sticker_boards에 penalty 필드 |
| F13 | 패널티 실행/완료 | 실패 시 패널티 내용 표시 + 완료 버튼 | penalty status: pending → completed |
| F14 | 홈 대시보드 | 진행 중 스티커판 목록, 최근 활동 | 쿼리: active 상태 boards + 최근 stickers |
| F15 | 기본 스티커 이미지 | 5~10종 기본 스티커 이미지 제공 | 앱 번들에 포함 |
| F16 | 푸시 알림 | 스티커 받음, 목표 달성, 만료 임박 알림 | Expo Notifications (FCM/APNs) |
