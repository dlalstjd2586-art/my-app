# 13. MVP 범위 축소안

## "이 정도면 출시 가능" 최소 범위

### 포함하는 것 ✅

| 기능 | 설명 | 이유 |
|------|------|------|
| 구글 로그인 | 카카오 대신 구글만 먼저 | 설정이 더 간단. 카카오는 2차에서 추가 |
| 닉네임 설정 | 프로필 사진은 나중에 | 최소한의 식별 정보 |
| 초대코드 연동 | 6자리 코드 생성 + 입력 | 핵심 기능 |
| 스티커판 생성 | 이름 + 목표 수 + 기간 + 보상텍스트 | 핵심 기능 |
| 스티커판 수락 | 상대방 동의 | 핵심 기능 |
| 스티커 주기 | 1개씩 부여 (메모 없이) | 핵심 인터랙션 |
| 진행률 표시 | 숫자 + 바 | 동기부여 |
| 목표 달성 처리 | 자동 상태 변경 + 보상 표시 | 핵심 기능 |
| 기본 스티커 3종 | 별, 하트, 엄지척 | 최소한의 선택지 |
| 홈 화면 | 진행 중 보드 1개 표시 | 메인 화면 |
| 기간 만료 처리 | **클라이언트에서 체크** (Cron 대신) | Cron 설정 복잡. 앱 열 때 만료 체크로 대체 |
| 보상 완료 버튼 | 텍스트 보상 + 완료 확인 | 최소 이행 추적 |

### 빼는 것 ❌ (2차로 미룸)

| 기능 | 빼는 이유 |
|------|---------|
| 카카오 로그인 | 카카오 개발자 설정이 복잡. 구글로 먼저 검증 |
| 애플 로그인 | Apple Developer 계정 필요. Play Store 먼저 출시하면 불필요 |
| 패널티 기능 | 있으면 좋지만 핵심은 아님. 보상만으로도 동작 |
| 푸시 알림 | 설정 복잡. 앱 내 알림 목록으로 대체 |
| 히스토리 화면 | 현재 진행 중 보드만 보여도 충분 |
| 배너 광고 | DAU 낮으면 수익 미미. UX 집중이 우선 |
| 커스텀 스티커 | 기본 3종으로 충분 |
| 스티커 메모 | 나중에 추가해도 됨 |
| 외부 링크 보상 | 텍스트 보상만으로 시작 |
| 설정 화면 | 로그아웃 버튼만 임시로 넣기 |
| 관계 해제 | 설정 화면에 포함. 최소 MVP에서는 생략 가능 |

---

## 축소된 MVP 화면 구성

```
[로그인] → [닉네임 입력] → [연동 화면]
                                ↓
                          [홈 대시보드]
                           ├── [스티커판 생성]
                           └── [스티커판 상세 + 스티커 주기]
```

**화면 수: 6개** (로그인, 닉네임, 연동, 홈, 생성, 상세)

---

## 축소 MVP 개발 예상 기간

| 단계 | 기간 | 내용 |
|------|------|------|
| 세팅 | 2일 | Expo + Supabase 프로젝트 생성, DB 마이그레이션 |
| 인증 | 2일 | 구글 로그인 + 닉네임 설정 |
| 연동 | 3일 | 초대코드 생성/입력/수락 |
| 스티커판 | 3일 | 생성 폼 + 수락 + 홈 표시 |
| 스티커 | 2일 | 부여 + 진행률 + 달성 처리 |
| 마무리 | 2일 | 만료 체크(클라이언트) + 보상 완료 + 기본 UI 다듬기 |
| 테스트 | 2일 | 실기기 테스트 + 버그 수정 |
| **합계** | **약 16일 (2~3주)** | |

## 축소 MVP의 제한사항 (사용자에게 고지)

| 제한 | 사용자 영향 | 해결 시기 |
|------|-----------|----------|
| 구글 로그인만 | 카카오 사용자 가입 불가 | 출시 후 1주 내 |
| 푸시 알림 없음 | 앱을 직접 열어야 확인 가능 | 출시 후 2주 내 |
| 스티커판 1개만 동시 진행 | 여러 챌린지 동시 불가 | 출시 후 1주 내 해제 |
| 히스토리 없음 | 과거 기록 확인 불가 | 출시 후 2주 내 |
| 패널티 없음 | 성공 보상만 가능 | 출시 후 2주 내 |

## "진짜 최소" 기준 체크리스트

아래 시나리오가 완성되면 MVP 출시 가능:

- [ ] 두 사람이 각각 앱에 가입할 수 있다
- [ ] 초대코드로 서로 연결할 수 있다
- [ ] 한 명이 스티커판을 만들고 상대방이 수락할 수 있다
- [ ] 스티커를 1개씩 줄 수 있다
- [ ] 몇 개 모았는지 확인할 수 있다
- [ ] 목표 달성 시 보상 내용이 표시된다
- [ ] 기간이 지나면 종료된다

**이 7가지가 되면, 나머지는 전부 업데이트로 추가 가능합니다.**
